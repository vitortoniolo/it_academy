Descrição do curso

Aprenda os fundamentos de arquitetura de software para sistemas de alta disponibilidade. Este curso abrange desde padrões básicos até microsserviços avançados, com uma abordagem prática e estrutural.

Microsserviços Mensageria Cloud Docker Alta disponibilidade

Módulos do curso

Módulo 1: Fundamentos6 aulas

Princípios de arquitetura, requisitos não-funcionais, escalabilidade vertical e horizontal e trade-offs essenciais.

Módulo 2: Monolitos vs Microsserviços8 aulas

Quando usar cada abordagem, decomposição de domínios, bounded contexts e estratégias de migração.

Módulo 3: Padrões de Mensageria7 aulas

Filas, tópicos, event-driven, idempotência, entrega garantida e padrões saga para transações distribuídas.

Módulo 4: Observabilidade & Deploy5 aulas

Logging, métricas, tracing distribuído, CI/CD e estratégias de deploy sem downtime.
